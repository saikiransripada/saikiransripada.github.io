---
title: "Get table structure in MySQL"
categories: [Database, MySQL]
---

SQL command to display a table structure.

{% highlight sql %}
EXPLAIN <table_name>;
{% endhighlight %}

OR

{% highlight sql %}
DESCRIBE <table_name>;
{% endhighlight %}

OR

{% highlight sql %}
DESC <table_name>;
{% endhighlight %}

You must have a database selected before you run these commands.
